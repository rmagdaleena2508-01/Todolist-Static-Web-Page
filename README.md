# 📋 Todolist Static Web Page

## 🚀 Overview

This project is a simple static **Todolist landing page** built using HTML and CSS.
It demonstrates fundamental front-end development concepts such as layout design, typography, background styling, and component structuring.

---

## ✨ Features

* Clean and minimal UI design
* Full-screen background image layout
* Center-aligned content structure
* Styled call-to-action button ("Start")
* Emphasis on readability and typography
* Underlined informational footer text

---

## 🛠 Tech Stack

* **HTML5** – Structure of the webpage
* **CSS3** – Styling and layout
* **Google Fonts** – Typography enhancement
